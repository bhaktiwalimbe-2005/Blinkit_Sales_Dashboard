# Blinkit_Sales_Dashboard
Blinkit Sales Analysis Dashboard (Power BI)
Tech: Power BI, DAX, Data Visualization
•	Designed an interactive sales dashboard using 5+ KPIs including Total Sales, Average Sales, Number of Orders, Outlet Count, and Customer Ratings.
•	Analyzed sales performance across Tier 1, Tier 2, and Tier 3 cities; identified that Tier 2 and Tier 3 outlets contributed higher combined revenue.
•	Evaluated outlet size performance and found medium-sized outlets generated the highest sales contribution.
